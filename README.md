# HIM — landing page

Public landing page for **HIM**, a local-LLM server administrator for Linux.

- Deterministic monitoring that never lies, plus a local LLM that explains *why*.
- Single file, zero dependencies (Python stdlib). Runs on your hardware. No API keys, no cloud.
- HIM never executes changes on its own — every fix is a human decision.

This repository contains **only the marketing page**. HIM itself is a commercial
product; its source is not published here.
